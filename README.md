AWS Services Used:
- Amazon EC2
- Amazon RDS (MySQL)
- Amazon S3
- IAM Role (EC2-S3-Role)
- Nginx
- Gunicorn

Deployment:
1. Created EC2 instance.
2. Configured Security Groups.
3. Attached IAM Role (EC2-S3-Role) for S3 access.
4. Connected Flask application to Amazon RDS.
5. Configured Nginx and Gunicorn.
6. Hosted application on EC2.
